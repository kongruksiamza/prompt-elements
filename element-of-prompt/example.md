## ตัวอย่าง Prompt (Example)
```
เขียนบทความเกี่ยวกับการท่องเที่ยว ตัวอย่าง "เชียงใหม่เมืองน่าเที่ยว อากาศเย็นสบาย วัฒนธรรมล้านนาน่าหลงใหล อาหารเหนือรสเลิศ" ช่วยเขียนบทความในลักษณะนี้เกี่ยวกับภูเก็ต
```

```
ช่วยเขียนสูตรทำขนมในรูปแบบนี้ 
ชื่อขนม: คุกกี้ช็อกโกแลตชิพ 
เวลาที่ใช้ : 30 นาที 
วัตถุดิบ: แป้ง 2 ถ้วย, เนย 1 ถ้วย 
ขั้นตอน: 1. ผสมแป้งกับเนย 
ช่วยเขียนสูตรบราวนี่ในรูปแบบเดียวกันนี้
```

```
แนะนำวิธีการตั้งเป้าหมายในชีวิตให้กับคนวัยทำงาน 
ตัวอย่าง : การตั้งเป้าหมายระยะสั้น เช่น การเรียนรู้ทักษะใหม่ๆ
```
```
แนะนำวิธีการเลือกสมาร์ทโฟนที่เหมาะสมสำหรับผู้สูงอายุ 
ตัวอย่าง: สมาร์ทโฟนที่มีหน้าจอขนาดใหญ่และฟังก์ชันที่เข้าใจง่าย
```
```
แนะนำเมนูอาหารเพื่อสุขภาพสำหรับผู้ที่ต้องการลดน้ำหนัก 
ตัวอย่าง: สลัดผักสดและไก่ย่างที่มีโปรตีนสูง
```
```
แนะนำวิธีการพัฒนาทักษะการเขียนโปรแกรมสำหรับผู้เริ่มต้น ตัวอย่าง: เริ่มจากการเรียนรู้ภาษา Python และทำโปรเจกต์เล็กๆ
```
```
เขียนอีเมลตอบกลับลูกค้าโดยใช้โครงสร้างเดียวกับตัวอย่างที่ให้ไว้ [แนบตัวอย่าง]
```
