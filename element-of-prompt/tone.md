## ตัวอย่าง Prompt (Tone)
```
เล่าเรื่องราวเกี่ยวกับวันแรกในมหาวิทยาลัยด้วยน้ำเสียงที่สนุกสนานและขบขัน
```
```
อธิบายวิธีเริ่มต้นลงทุนในน้ำเสียงที่ผ่อนคลายและให้กำลังใจ
```
```
เขียนบทกลอนสั้นๆ เกี่ยวกับความรักในโทนเสียงที่โรแมนติก
```
```
เขียนอีเมลขอบคุณลูกค้าด้วยน้ำเสียงที่อบอุ่นและจริงใจ
```
```
เขียนบทความเกี่ยวกับภาวะโลกร้อนในโทนที่จริงจังและให้ข้อมูลเชิงลึก
```
```
บรรยายเกี่ยวกับพระอาทิตย์ตกดินในโทนดราม่า
```
```
เขียนเรื่องสั้นเกี่ยวกับการผจญภัยของแมวจรในเมืองใหญ่ ใช้น้ำเสียงสนุกสนาน
```
```
เขียนข้อความให้กำลังใจเพื่อนที่กำลังเครียดเรื่องงาน ใช้น้ำเสียงอบอุ่นและจริงใจ
```
```
เขียนคำคมสร้างแรงบันดาลใจให้คนที่กำลังท้อแท้ ใช้น้ำเสียงมั่นใจ กระตือรือร้นและเต็มไปด้วยพลัง
```
